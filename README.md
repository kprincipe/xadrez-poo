# Projeto POO - Xadrez
Xadrez com as peças funcionais e implementações de movimentos.
> [!IMPORTANT]
> Projeto final da turma de 2° período de Programação Orientada a Objetos, do curso TADS no semestre 2026/1

## Como mandar sua parte do projeto
1. Crie um FORK desse repositório
2. Clone seu FORK e abra ele no Eclipse seguindo os passos:
> Import > Git > Projects from Git > Existing local repository > Add (Adicione a pasta referente ao repositório) > Finish
3. Faça suas modificações e mantenha elas atualizadas na `master`
4. Dê PUSH nas suas mudanças e mande elas para seu FORK remoto
5. Crie um PULL REQUEST no seu FORK e mande para a BRANCH da sua equipe neste repositório
